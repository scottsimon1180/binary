# Repository Guidelines

## Project Structure & Module Organization

This repository is a static Binary Translator web app.

- `index.html` contains markup, inline SVG controls, metadata links, and panel structure.
- `style.css` contains theme tokens, responsive layout, and all visual styling.
- `script.js` contains conversion logic, settings persistence, rendering, and event handling.
- `manifest.json` defines PWA metadata and icon references.
- `icon.svg` is the app icon.
